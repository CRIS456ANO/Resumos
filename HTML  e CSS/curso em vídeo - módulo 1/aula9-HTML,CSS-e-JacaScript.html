A diferença entre HTML, CSS e JavaScript

Este resumo é fruto de um estudo da aula 9 do primeiro módulo do curso de HTML 5 e CSS 3 feito através do canal no youtube do Gustavo guanabara.

A (linguagem) HTML e as (folhas de estilo em cascata) CSS

É errado dizer: "Eu programo em HTML e CSS" porque não são linguagens de programação, o correto é dizer: "Eu desenvolvo (crio) em HTML e CSS".

Conteúdo em HTML:

<h1&>Exemplo de título</h1>
<p>Exemplo de paragrafo</p>
<img src="foto.png" alt="exemplo de foto"> (não possui fechamento de tag)

Conteúdo em CSS:

h1 {
font-family:Arial;
font-size:20 pt;
color:blue;
}

Estrutura básica do documento HTMl:

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Aula 9 - capítulo 3, aula 1</title>
</head>
<body>
<h1>Olá, mundo!</h1>
